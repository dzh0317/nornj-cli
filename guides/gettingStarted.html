# 快速起步

本节我们将使用NornJ-cli快速创建一个可以在浏览器中运行的项目脚手架。

# 安装CLI脚手架

输入命令：

npm install -g nornj-cli      # mac下需要在前面加sudo命令

# 使用CLI脚手架初始化react + mobx-state-tree的前端项目

输入以下命令，会在当前目录创建新项目初始代码：

nj init

Please select a project template:
> react-mst-universal  # 请选择项目模板
  react-mst-app
  react-mst
  react-mobx
  other

Project Name: react-mst-test           # 输入新创建项目的目录名称
Do you want to npm install(Y/N)?       # 选择是否安装npm包

输入y后，按回车键就会自动安装项目依赖的各npm包，大概等几分钟就可以装好：

# 运行项目

用cd react-mst-test命令进入我们刚创建好的项目目录，然后输入命令：

npm run dev-web

然后会开始运行webpack构建过程，构建成功后请在浏览器输入地址：localhost:8080/dist/web即可查看页面：

# 添加新页面

在react-mst-test目录下输入如下命令：

nj ap

# 请选择要生成的页面类型
What type of page do you want to generate ?
  default  # 增删改查
> chart    # 图表
  empty    # 空页面

# 输入新创建的页面名称
Page Name: newChartPage

输入y后，按回车键就会立即创建好一个新建的图表页面，且各子目录下的newChartPage文件都已创建好了：

# 运行新添加的页面

打开react-mst-test/src/stores/rootStore.js文件，将这里的第一个菜单中的第一个页面名称Page1_1改为NewChartPage：

sider: types.optional(SiderStore, {
  isOpen: false,
  current: 'page1',
  menuData: [{
    type: 'group',
    index: 'Menu1_1',
    name: '一级菜单1',
    expanded: false,
    children: [{
      type: 'group',
      index: 'Menu2_1',
      name: '二级菜单1',
      expanded: false,
      children: [
        { type: 'item', level: 3, link: '/NewChartPage', index: 'NewChartPage', name: '页面1-1' },
        { type: 'item', level: 3, link: '/Page1_2', index: 'Page1_2', name: '页面1-2' },
      ]
    }, 
    ...
  }, 
  ...]
}),

再次运行命令进行构建：

npm run dev-web

然后在浏览器输入地址：localhost:8080/dist/web查看页面，新创建的图表页已展示出来了：
