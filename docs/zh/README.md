## 起步

安装：

``` bash
npm install -g nornj-cli
# OR
yarn global add nornj-cli
```

创建一个项目：

``` bash
nj init

# 请按上下键选择项目模板
Please select a project template:
> react-mst
  react-mst-app
  react-mobx
  other

# 输入新建的项目目录名称
Project Name: my-project
```
