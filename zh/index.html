NornJ CLI

🛠️ 基于 NornJ 模板引擎的脚手架工具

起步 →

开箱即用

以 CLI 形式对 React、Mobx、Babel、TypeScript、ESLint、单元测试和 End-to-end 测试提供开箱即用的支持。

可配置化

完全基于模板引擎设计，通过预设各种配置参数，从而生成适用于不同用户需求的各类代码文件。

易于扩展

它的资源模板系统允许用户根据自身喜好，依项目、页面、组件等多种维度定制出适合的项目模板。

# 起步

安装：

npm install -g nornj-cli
# OR
yarn global add nornj-cli

创建一个项目：

nj init

# 请按上下键选择项目模板
Please select a project template:
> react-mst
  react-mst-app
  react-mobx
  other

# 输入新建的项目目录名称
Project Name: my-project
